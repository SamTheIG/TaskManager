# TaskManager
ok for using this thing:
* install python if you don't have it
* in the terminal or CMD type these lines:
    >   `pip3 install -r requirements.txt`
    >   `python3 -m venv venv`
    >    * now for linux/mac:
    >        `source venv/bin/activate`
    >    * for windows CMD:
    >        `IDK`
    >    `pip3 install flask, flask_login`
    >    `python3 app.py`
